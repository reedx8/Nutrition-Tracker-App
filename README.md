# Nutrition-Tracker-App
iOS app made with React native, managed by Expo.

Dependancy list: TBD

## To run app:
  1. `expo build:ios` to compile.
  2. `nodemone server` in ./backend to start backend server
  2. `expo start --ios` in ./frontend to run app
